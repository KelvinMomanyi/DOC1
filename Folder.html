The Royal Hotels React Template Folder structure is made up of several key files and folders namely:

  • Public Folder

    Contains the logo's location, which should preferably be in svg

  • Src Folder

    This is the location of most of the files including the components, context & Pages folder

    • Components

      This folder contains all reusable components that are in the template

    • Context

      This folder contains the files which utilize the Context API for state managment in the whole template.

    • Pages

      The folder contains the different pages in the react template. Changing the contents of page should start from here

  • App.jsx

    This is the main file which will handle the routing of the pages, if you want to change the routing of a page or want to add a page, here is where you will define its routing.

  • Main.jsx

    This is the index file of the react template, no need of changing anything on this file because it is already setup and ready for use.

  • index.css

    This is the main styling file which is written in css and tailwind in css, it can be used to add styling where using tailwind may fall short

  • Others

    • data.jsx

      The file that contains dummy data for the template, you might decide to edit it or even get rid of it incase you are intergrating the template with your own backend

    • datepicker.css

      contains the styling for the datepicker component depedency that is installed for the purposes of setting a booking on the hotel

    • tailwind.config.js

      This is the main configuration file for tailwind and is useful when configuring fonts, primary and secondary colors, background images among others
