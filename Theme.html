Theme

Color

By default the theme set for this template is:

  • primary:'#0a0a0a', Black background color
  • accent: '#967142', Gold/brown color
  • secondary:'#991B1B', Dark red color

These main theme colors can be changed on the tailwind.config.js file in the secrion as can be seen below

Fonts

The fonts used in this template are from Google fonts and have being imported via the index.html file in the main folder. In terms of the types, the template uses three major fonts, namely:

  • Primary:'Gilda Display'
  • Secondary:'Barlow'
  • Tertiary:'Barlow Condensed'

Customization

  • Step 1

    Link the fonts of your choice from google fonts or any other sources using the Link tag in the main index.html folder

  • Step 2

    In the tailwind.config.js file in the main folder, change the fonts to your preferred font types in the 'fontFamily' category

Headings, Titles & Content

In the Royal Hotels React template, all headings and titles can be easily customized according to one's liking by following these steps:

  • Locate Heading & Title Components

    Identify the components that are responsible for rendering the particular heading or title that needs to be changed. They may be easily be identified in their respective tags such as h1,h2,h3, p, span and a tags.

  • Identify Component props or styling

    Review the props available for each heading tag. These props might include attributes such as color, fontfamily, fontweight, textAlign, size etc.

  • Modify props

    To customize a heading, simply locate its corresponding tag and adjust its props as desired. For instance if you want to change the color of a text to blue, locate its tag in the compoment and change it prop to 'text-blue-500', according to your preference. Or change its content by changing the content between the tags for instance Home to About
