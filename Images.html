Images

Images used in the Royal Hotels Template varies with certain images being available in the Assets folder while other beimg direct image address links.

These main theme colors can be changed on the tailwind.config.js file in the secrion as can be seen below

  • Assets Folder

    Contains three folders of images which include:

    • heroSlider

      Contains images that are in the main Carousel on the home page. Incase you want to change the images on the carousel, the images in this folder ar what you will change.

    • rooms

      Contains all the images of the hotel rooms in the template, customizing the images of the rooms will require you to do it from here or ignore uncae you are using a database.

  • Others

    Other image sources occur as direct links to the image adress and can be changed on the individual components
