Fonts

The fonts used in this template are from Google fonts and have being imported via the index.html file in the main folder. In terms of the types, the template uses three major fonts, namely:

  • Primary:'Gilda Display'
  • Secondary:'Barlow'
  • Tertiary:'Barlow Condensed'

Customization

  • Step 1

    Link the fonts of your choice from google fonts or any other sources using the Link tag in the main index.html folder

  • Step 2

    In the tailwind.config.js file in the main folder, change the fonts to your preferred font types in the 'fontFamily' category
